MINIMIAM


El proyecto trato de crear una serie de fotografías a partir de diferente material que se tienen en casa, y se pueda usar como fondo o parte del escenario que servirá para recrear varias escenas de películas, y que se pueda observar que se logran buenas fotografías creativas con elementos que están a la mano y puedan despertar la imaginación utilizando los elementos para crear la composición de cada foto. Dichas fotos tienen como inspiración a el fotógrafo Pierre Javalle y Akiko Ida.
